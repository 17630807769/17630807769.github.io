Vue单文件组件

基本用法

  • 以.vue结尾
  • template：组件的模块区域
  • script：业务逻辑区域
  • style：样式区域

在webpack中使用vue

  1. 运行npm i vue -S安装
  2. 在src/index.js中通过import Vue from ‘vue’导入
  3. 创建vue的实例对象，并指定要控制的el区域
  4. 通过render函数渲染APP组件

Vuex

基本介绍

  • Vuex是实现组件全局状态(数据)管理的一种机制，可以方便的实现组件之间数据的共享
  • Vuex中的数据都是响应式的，能够实时保持数据与页面的同步

使用

  1. 安装vuex依赖包npm install vuex –save

  2. 导入vuex包

  3. 创建store对象
